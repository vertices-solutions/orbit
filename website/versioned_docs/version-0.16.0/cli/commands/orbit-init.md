Initialize a project directory with an Orbitfile

Parent command: [`orbit`](./orbit)

**Usage:** `orbit init [OPTIONS] <PATH>`

## Arguments

- `<PATH>`

## Flags and Options

- `--name <NAME>`: Project name - it will be stored in Orbitfile and you will be able to filter jobs by it
- `--non-interactive`: Run without prompts, fail on MFA, and output JSON only.
- `-h, --help`: Print help
