Build a project tarball and register it locally

Parent command: [`orbit project`](./orbit-project)

**Usage:** `orbit project build [OPTIONS] <PATH>`

## Arguments

- `<PATH>`

## Flags and Options

- `--package-git`: Include .git directory in the tarball
- `--non-interactive`: Run without prompts, fail on MFA, and output JSON only.
- `-h, --help`: Print help
