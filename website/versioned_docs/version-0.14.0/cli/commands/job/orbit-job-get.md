Show job details

Parent command: [`orbit job`](./orbit-job)

**Usage:** `orbit job get [OPTIONS] <JOB_ID>`

## Arguments

- `<JOB_ID>`: Job id from the daemon

## Flags and Options

- `--cluster <CLUSTER>`
- `--non-interactive`: Run without prompts, fail on MFA, and output JSON only.
- `-h, --help`: Print help
