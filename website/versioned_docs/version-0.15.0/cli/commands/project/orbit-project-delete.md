Delete a project from the local registry

Parent command: [`orbit project`](./orbit-project)

**Usage:** `orbit project delete [OPTIONS] <NAME>`

## Arguments

- `<NAME>`: Project name or name:tag

## Flags and Options

- `-y, --yes`: Skip the confirmation prompt
- `--non-interactive`: Run without prompts, fail on MFA, and output JSON only.
- `-h, --help`: Print help
