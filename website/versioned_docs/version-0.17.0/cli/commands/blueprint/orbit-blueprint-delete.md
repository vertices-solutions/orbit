Delete a blueprint from the local registry

Parent command: [`orbit blueprint`](./orbit-blueprint)

**Usage:** `orbit blueprint delete [OPTIONS] <NAME>`

## Arguments

- `<NAME>`: Blueprint name or name:tag

## Flags and Options

- `-y, --yes`: Skip the confirmation prompt
- `--non-interactive`: Run without prompts, fail on MFA, and output JSON only.
- `-h, --help`: Print help
