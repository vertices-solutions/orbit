Build a blueprint tarball and register it locally

Parent command: [`orbit blueprint`](./orbit-blueprint)

**Usage:** `orbit blueprint build [OPTIONS] <PATH>`

## Arguments

- `<PATH>`

## Flags and Options

- `--package-git`: Include .git directory in the tarball
- `--non-interactive`: Run without prompts, fail on MFA, and output JSON only.
- `-h, --help`: Print help
