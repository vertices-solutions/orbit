Connect to a cluster and validate the SSH session

Parent command: [`orbit cluster`](./orbit-cluster)

**Usage:** `orbit cluster connect <NAME>`

## Arguments

- `<NAME>`: Cluster name to connect

## Flags and Options

- `--non-interactive`: Run without prompts, fail on MFA, and output JSON only.
- `-h, --help`: Print help
