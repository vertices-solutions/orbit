List clusters

Parent command: [`orbit cluster`](./orbit-cluster)

**Usage:** `orbit cluster list [OPTIONS]`

## Flags and Options

- `--check-reachability`: Perform reachability checks and include the reachable column
- `--non-interactive`: Run without prompts, fail on MFA, and output JSON only.
- `-h, --help`: Print help
