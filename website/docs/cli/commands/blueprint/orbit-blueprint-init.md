Initialize a blueprint root and Orbitfile

Parent command: [`orbit blueprint`](./orbit-blueprint)

**Usage:** `orbit blueprint init [OPTIONS] <PATH>`

## Arguments

- `<PATH>`

## Flags and Options

- `--name <NAME>`: Blueprint name stored in Orbitfile and the local registry
- `--non-interactive`: Run without prompts, fail on MFA, and output JSON only.
- `-h, --help`: Print help
