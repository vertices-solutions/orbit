Initialize a project root and Orbitfile

Parent command: [`orbit project`](./orbit-project)

**Usage:** `orbit project init [OPTIONS] <PATH>`

## Arguments

- `<PATH>`

## Flags and Options

- `--name <NAME>`: Project name stored in Orbitfile and the local registry
- `--non-interactive`: Run without prompts, fail on MFA, and output JSON only.
- `-h, --help`: Print help
